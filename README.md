# 🛒 Blinkit Sales Dashboard

## 📌 Project Overview

This project analyzes Blinkit's sales performance using Power BI. The dashboard provides insights into sales trends, outlet performance, product categories, customer ratings, and item distribution to support business decision-making.

---

## 🎯 Business Problem

Retail businesses need to understand sales performance across different outlet types, item categories, and customer preferences.

This dashboard helps answer:

- Which outlet generates the highest sales?
- Which product category performs best?
- How do outlet sizes impact sales?
- Which fat content category sells the most?
- What are the customer rating trends?

---

## 📈 Dashboard Features

- Total Sales
- Average Sales
- Number of Items
- Average Rating
- Sales by Item Type
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Fat Content
- Interactive Filters
- KPI Cards

---

## 📊 Key Insights

- Total Sales exceeded **$1.2M**.
- Supermarket outlets generated the highest revenue.
- Regular-fat products contributed more sales than low-fat products.
- Customer ratings remained consistently high across outlets.
- Tier-wise outlet analysis revealed differences in performance.
